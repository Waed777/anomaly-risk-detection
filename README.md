# Financial Anomaly & Risk Detection System

This project demonstrates a machine learning-based anomaly and risk detection system
designed for institutional and financial environments.

## Business Value
- Detect unusual transactions
- Reduce operational and financial risk
- Support management decision-making

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Features
- Synthetic transaction data generation
- Anomaly detection using Isolation Forest
- Risk scoring and alerts dashboard

## How to Run
```bash
streamlit run app.py
